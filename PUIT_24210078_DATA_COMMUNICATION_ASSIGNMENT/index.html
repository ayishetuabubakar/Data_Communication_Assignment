Understanding the Fundamentals

About Data Communication

Data communication is the process of transferring information from one device to another through a guided or wireless medium using agreed rules called protocols.

01

Exchange of Data

Communication begins when a sender creates a message and transmits it to a receiver through a physical cable, radio signal, fiber link, or other channel.

02

Shared Resources

Networks allow users to share files, printers, applications, internet access, databases, and cloud services across different locations.

03

Rules and Standards

Protocols make communication reliable by defining how data is formatted, addressed, routed, delivered, checked, and protected.

Core Concepts and Components

Introduction to Data Communication

Every communication system depends on five connected parts: sender, message, medium, receiver, and protocol. When these parts work together, data can move accurately and efficiently.

Sender Creates the data
Message Text, audio, video, or files
Medium Cable, fiber, or wireless link
Receiver Gets and interprets the data
Protocol Rules that guide the exchange

Networking Devices

Hardware devices make it possible for computers and networks to connect, forward, and manage data.

Router

Connects different networks and forwards packets using IP addresses.

Layer 3

Switch

Connects devices inside a local area network and forwards frames using MAC addresses.

Layer 2

Modem

Converts signals between a local network and an internet service provider connection.

Signal conversion

Access Point

Provides wireless network access for laptops, phones, printers, and other Wi-Fi devices.

802.11 wireless

Gateway

Acts as the path between separate networks and can translate between different protocols.

Network edge

Repeater

Regenerates weak signals so they can travel farther without losing quality.

Signal boost

Transmission Media

Transmission media are the paths that carry data from source to destination.

Twisted Pair Cable

Affordable and common in LANs, with typical runs of about 100 meters.

Coaxial Cable

Offers strong shielding and has been widely used for cable television and older networks.

Fiber Optic Cable

Uses pulses of light for very high speed, long distance, and strong resistance to interference.

Wireless Media

Radio, microwave, infrared, and satellite links support mobility and remote communication.

Protocols & IP Addressing

Protocols organize communication while IP addresses identify devices on a network.

TCP/IP Reliable packet delivery and routing across networks.
HTTP/HTTPS Web communication, with HTTPS adding encryption.
DNS Translates domain names into IP addresses.
DHCP Automatically assigns IP addresses to devices.
IPv4 / IPv6 32-bit addressing compared with newer 128-bit addressing.

Protecting Data and Systems

Network Security

Network security keeps communication private, reliable, and protected from unauthorized access. Good security combines tools, policies, updates, and responsible user behavior.

Firewall Encryption VPN Authentication Backups Security updates

Historical Foundation

Telegraph Communication System

The telegraph was one of the first electrical systems used for long-distance communication. Operators encoded messages into Morse code, sent electrical pulses through wires, and decoded the received signal into readable text.
